# ゲームデザインドキュメント

## タイトル（仮）：Dungeon Liberate ― クロウネ奪還

> 3Dダンジョン探索 × 陣地管理（ターン制）
> モンスターの生態系を読み、地下交易都市クロウネを人間族の手に取り戻せ。

---

## コンセプト

かつて繁栄した地下交易都市クロウネは、汚染されたクリスタルがモンスターを引き寄せ廃墟と化した。
プレイヤーは傭兵団のリーダーとして、左上の旧人間居住区を拠点に3つの敵エリアへ侵攻する。
モンスターの**クリスタルを封印**し、すべてのクリスタルを制圧することが最終ゴール。
人間族の拠点がモンスターに占拠されるとゲームオーバー。

---

## 物語・世界観

かつて大陸の地下に栄えた交易都市「クロウネ」。この都市が繁栄できたのは、地下深部に眠る「クリスタル」のおかげだった。「王のクリスタル」は都市全体に魔力エネルギーを供給する動力源であり、人々はそれを守り、崇めてきた。

しかし100年前、暗黒魔術師の侵攻によって「クリスタル」は汚染された。制御を失ったクリスタルは闇の魔力を発し始め、各地からモンスターを引き寄せてしまう。今では3つの種族がクリスタルを奪い合い、都市は壊滅状態に。

地上の王国は地下航路を断たれ疲弊。布告を出す。**「クロウネを奪還した者に、永代統治権を与える」**

プレイヤーは、金と名誉を求めて集まった傭兵団のリーダー。左上の旧人間居住区を拠点に、3つの敵エリア（ゴブリン・リザード・オーガ）を制圧しながら、すべてのクリスタルを封印することを目指す。

## 核心メカニクス：スポーナーシステム

### スポーナーとは
- 各モンスター陣地の中心に1つ存在する構造物
- 定期的にそのスポーナーの種族のモンスターをスポーンし続ける
- スポーナーの種族が変わると、スポーンされるモンスターも変わる

### 封印
- スポーナーには必ず**守衛モンスターが1体**待機している
- その守衛を倒すとスポーナーを封印できる
- 封印するとモンスターのスポーンが停止する

### 奪還（再解放）
- 封印されたスポーナーに別種族のモンスターが接触すると、その種族のスポーナーに変換される
- 封印は永続ではなく、守備が必要

### スポーナー変換の連鎖
```
[ゴブリン族スポーナー] → プレイヤーが封印
        ↓
[封印済みスポーナー] ← 魔族モンスターが接触
        ↓
[魔族スポーナー] ← 魔族モンスターがスポーンし始める
```

## 6種族設計

### ユニット供給方法

| 陣営 | 供給方法 |
|---|---|
| 人間・エルフ・ドワーフ | Goldを消費して雇用（雇った分だけ増える） |
| ゴブリン・リザード・オーガ | クリスタルから自動スポーン |

### ステータス一覧

| 種族 | HP | ATK | ターン移動 | 備考 |
|---|---|---|---|---|
| 👺 ゴブリン | 30 | 8 | 1マス | 数の暴力。スポーン間隔が短い |
| 🦎 リザード | 60 | 15 | 1マス | バランス型。中程度のスポーン |
| 👹 オーガ | 120 | 30 | 移動か攻撃のどちらか | 強力だがスポーン間隔が長い |
| 👤 人間 | 80 | 18 | 1マス | 汎用型 |
| 🧝 エルフ | 50 | 25 | 2マス | ガラスの大砲。2マス移動可能 |
| ⛏ ドワーフ | 120 | 12 | 移動か攻撃のどちらか | 鉄壁の盾 |

> **SPD廃止について**: 最終仕様はターン制のため「移動速度（マス/秒）」は廃止。代わりに「1ターンに何マス動けるか」で個性を表現する。エルフは2マス移動、ドワーフ・オーガは移動と攻撃を同一ターンに行えない制約で差別化する。

### スポーン間隔（モンスター陣営のみ）

| 種族 | 間隔の目安 | 理由 |
|---|---|---|
| ゴブリン | 短い（6秒相当） | 弱い分、数で補う |
| リザード | 中（10秒相当） | バランス型 |
| オーガ | 長い（20秒相当） | 強力なため頻繁には出ない |

クリスタル所有数が多い勢力ほどスポーンが加速する（多拠点ボーナス）。

## 6種族の相性システム

### モンスター陣営の三すくみ

```
オーガ → ゴブリン → リザード → オーガ
```

| 勝敗 | 理由 |
|---|---|
| **オーガ > ゴブリン** | 圧倒的な力でゴブリンの大群を薙ぎ払う |
| **ゴブリン > リザード** | バランス型のリザードは数の暴力に対応しきれない |
| **リザード > オーガ** | 集団の連携と機動力でオーガ個体を封じ込める |

### 人間陣営の相性

```
エルフ → オーガ → ドワーフ → ゴブリン → 人間 → リザード → エルフ
```

| 種族 | 強い相手 | 理由 | 弱い相手 | 理由 |
|---|---|---|---|---|
| **人間** | リザード | 戦術の多様性が本能で動くリザードの裏をかく | ゴブリン | 数の暴力で包囲・消耗させられる |
| **エルフ** | オーガ | 俊敏さと精度でオーガの鎧の隙間を突く | リザード | 爬虫類の反射神経がエルフの速度に対応、鱗が矢を弾く |
| **ドワーフ** | ゴブリン | 厚い鎧がゴブリンの貧弱な武器を無効化 | オーガ | オーガの怪力が鎧ごと吹き飛ばす |

### 全体相性マップ

```
      人間族陣営                    モンスター陣営
  ┌───────────────────┐         ┌─────────────────────┐
  │ 人間  ──勝→ リザード ──勝→ オーガ ─┐ │
  │  ↑負           ↑負         ↑負      │ │
  │ゴブリン ←勝── エルフ  ←勝── ドワーフ ←┘ │
  └───────────────────┘         └─────────────────────┘
```

プレイヤーは攻略するエリアの支配種族を確認し、**どの人間族NPCを雇うか**を選ぶ戦略判断が生まれる。

## バトルシステム

### ダメージ計算式

```
ダメージ = ATK × (0.8〜1.2のランダム) × 相性補正
```

### 相性補正

- **有利な組み合わせ**：補正なし（×1.0）
- **不利な組み合わせ**：ATK × 0.7（攻撃側の与ダメージが減少）

> 実装方針：デバフは防御側ではなく**攻撃側**にかける（シンプルな実装のため）。

### 相性補正対応表

不利な組み合わせのみ補正を適用（ATK × 0.7）。記載のないペアはすべて × 1.0。

| 攻撃側 | 防御側 | 補正 | 理由 |
|---|---|---|---|
| 人間 | ゴブリン | ×0.7 | ゴブリン > ヒューマン |
| エルフ | リザード | ×0.7 | 鱗が矢を弾く |
| ドワーフ | オーガ | ×0.7 | 怪力が鎧ごと吹き飛ばす |
| ゴブリン | オーガ | ×0.7 | 三すくみ：オーガ > ゴブリン |
| リザード | ゴブリン | ×0.7 | 三すくみ：ゴブリン > リザード |
| オーガ | リザード | ×0.7 | 三すくみ：リザード > オーガ |

## モンスター生態系

### 陣地とエリア
- 各スポーナーを中心に **5×5マスのエリア**を陣地とする
- 陣地内のモンスター配置：
  - スポーナー：守衛1体が常駐
  - エリア内：徘徊モンスター（通常4体未満で安定）

### 溢れ出しメカニクス（閾値システム）
```
エリア内のモンスター数 ≥ 5（守衛1体 + 徘徊4体以上）
        ↓
閾値超過 → エリア外へ溢れ出す
        ↓
  ┌─────────────────────────────┐
  │ 近くに封印済みスポーナーあり → 奪還へ向かう  │
  │ なし → 人間族拠点へ向かう               │
  └─────────────────────────────┘
```

### モンスター同士の戦闘
- 異種族モンスターが接触すると**自動的に戦闘が発生**
- プレイヤーは戦闘画面を見なくてもよい
- ダンジョン全体図にモンスター数が常時表示される
- 戦闘発生と結果はメッセージログで確認できる

```
例：「ゴブリン族 vs 魔族が戦闘！ → ゴブリン族が勝利 / 魔族が1体減少」
```

### 生態系を利用した攻略
- モンスター同士を戦わせて数を減らし、閾値を下げる
- 弱ったところで漁夫の利を取りスポーナーを封印する
- 陣地争いの流れを読み、封印の優先順位を判断する

## プレイヤーの行動

### ダンジョン探索（3Dビュー）
- レイキャスティング方式のファーストパーソン視点
- グリッドベースの1マス単位移動（ターン制）
- 探索済みエリアはダンジョン全体図に反映される

### 戦闘（Wizardry風コマンドバトル）
- モンスターと接触するとバトル画面へ遷移
- コマンド選択式のターン制バトル
- 守衛を倒すことでスポーナーの封印が可能になる

### Gold収集
- 戦闘・探索によりGoldを獲得
- GoldはNPCハンターの増員・装備強化に使用

---

## NPCハンターシステム

### 概要
- プレイヤーがGoldを消費してNPCハンターを雇用・派遣できる
- NPCは自律的に行動し、プレイヤーの指示で役割を切り替えられる

### 指示できる役割

| 役割 | 条件 | 行動内容 |
|---|---|---|
| **守備** | 封印済みスポーナーが存在する | 指定スポーナーを守り、奪還を阻止する |
| **探索** | 常時可能 | 未踏エリアへ向かい、マップ開拓・Gold回収を行う |

### 運用戦略例
```
NPC_A → [守備] スポーナー(3,7)を守る
NPC_B → [探索] 未踏エリアへ向かう
NPC_C → [守備] スポーナー(9,2)を守る
```

---

## 成長・強化システム

### 装備・武器ビルド構築（主要やりこみ軸）
- モンスターを倒すと素材やGoldを獲得
- 装備・武器を強化・組み合わせてビルドを構築する
- ビルドの最適化が攻略効率に直結する

### NPC増員
- Goldを消費してNPCハンターを追加雇用
- NPCが増えることで、複数スポーナーを同時防衛できるようになる

## ゲームループ全体図

```
【ゲーム開始】
  ↓
人間族拠点に帰還・状況確認
  ├─ ダンジョン全体図で陣地状況を確認
  ├─ NPCへの指示を更新（守備/探索）
  └─ Goldで装備強化・NPC増員
  ↓
プレイヤーがダンジョンへ潜入（3D探索）
  ├─ モンスター同士の戦闘結果を確認
  ├─ 生態系の流れを読んで封印順を判断
  ├─ 守衛を倒してスポーナーを封印
  └─ Gold・素材を回収して帰還
  ↓
【繰り返し】
  ↓
全スポーナー封印 → ダンジョン解放 → CLEAR！
  または
人間族拠点が占拠される → GAME OVER
```
