| 種族 | 間隔 | 備考 |

| ゴブリン | 10秒 | 数の暴力防止のため7秒から延長 |

### マップ

- DFS + ループ50本の手続き的迷路生成
- 生成後に袋小路削減パスを実行（通路が1方向のみのセルの隣接壁をランダムに1つ開放）
- 4隅に各勢力ゾーン（人間：左上、ゴブリン：右上、リザード：左下、オーガ：右下）
- 中央に中立ゾーン

### ユニット挙動アルゴリズム

全陣営のユニットはすべて自動制御です。人間族・ゴブリン・リザード・オーガで**アルゴリズムは共通**であり、違いはスポーン間隔・ユニット種のみです。

#### 優先度つき行動ツリー（毎フレーム評価）

| 優先度 | 条件 | 行動 |
|--------|------|------|
| 1 | 同一セルに敵がいる | 敵をランダムに1体選択して攻撃（DPS）。他の行動はスキップ |
| 2 | 回復フラグON | 自軍クリスタルで回復（MaxHP÷8 HP/秒）。満タンで通常状態へ |
| 3 | 退却フラグON | 最寄りの自軍クリスタルへBFS移動 |
| 4 | HP ≤ MaxHP/3 かつ非交戦 | 退却フラグをONにし最寄りの自軍クリスタルへ向かう |
| 5 | 現在セルが敵・中立クリスタル | 即時占領し、ランダムに次のターゲットを設定 |
| 6 | 目標クリスタルが自陣になった | ランダムに再ターゲット |
| 7 | 常時 | 目標クリスタルへBFS経路を3秒ごとに再計算し、1/spd 秒ごとに1マス移動 |

#### ターゲット選択ルール

- **スポーン時**：自陣以外のクリスタルを**距離反比例の重み**でランダムに1つ選択（近いほど選ばれやすい。重み = 1/(距離+1)）
- **途中変更**：なし（目標クリスタルへ向かい続ける）
- **強制再ターゲット**：目標クリスタルの占領成功時・目標が自陣になったとき → 距離重みつきランダムで再設定
- **ターゲットがnullの場合**：最近傍の敵クリスタルをフォールバックとして使用

#### スポーン

- **人間族**：全クリスタルから 🗡 人間 → 🏹 エルフ → ⛏ ドワーフ のローテーションで自動スポーン（10秒 CD）
- **他種族**：各クリスタルが種族別間隔ごとに単一種をスポーン
- **共通**：勢力ユニット上限（所有クリスタル数 × 5）に達している場合はスポーンしない
- クールダウン中は💎に暗いオーバーレイ＋白い扇形で残り時間を表示

### 退却・回復システム

- **退却トリガー**：非交戦状態で HP ≤ MaxHP / 3 になった場合、最寄りの自軍クリスタルへ自動退却
- **回復速度**：MaxHP ÷ 8 HP/秒（全種族ともに1/3HPから満タンまで約5秒で統一）
- **戦闘割り込み**：回復中に敵が同セルに来た場合、即座に戦闘状態へ移行し回復を中断
- **退却先消失**：退却中に目標クリスタルが敵に占領された場合、次の最寄り自軍クリスタルへ切り替え
- **自軍クリスタルなし**：退却先が存在しない場合は通常行動（敵クリスタルを目標）に戻る

### ログシステム

- ゲーム中は以下のデータを自動収集し localStorage に保存（30秒ごと）
  - ユニット統計（種族別：スポーン数・キル数・撃破数・平均生存時間・退却数・回復成功数・退却中死亡数）
  - マッチアップ統計（攻撃種族 × 防御種族のキル数）
  - 30秒ごとの勢力スナップショット（クリスタル数・ユニット数）
  - 勢力別スポーン総数
- **📥 ログをDL & クリア** ボタンで JSON ファイルをダウンロードし、キャッシュを自動クリア
- **🔄 リスタート** ボタンでもキャッシュをクリア（ダウンロードなし）
- ログファイルは `logs/` フォルダに格納して管理

## 検証済みの知見

| 知見 | 詳細 |
|---|---|
| スポーン上限22は過密になりやすい | 上限を撤廃しても動作は安定。数の均衡はスポーン間隔で制御すべき |
| 8秒均一スポーンは全勢力が拮抗しすぎる | 種族別間隔に変更済み。さらにゴブリン14→7s、リザード20→12sへ短縮 |
| AI全員が同じクリスタルに集中する | ランダムターゲット割り当て＋10〜20秒の再評価タイマーで分散。勢力が均等に拡散するようになった |
| 袋小路が多いと一部エリアに孤立するユニットが発生 | 袋小路削減パスで通行可能性が向上。ユニットが行き止まりに詰まる頻度が減少 |
| 人間ユニットがスポーン後しばらく動かない | `targetCrystal: null` に対し `!== undefined` チェックが誤判定。truthy チェックに修正し即時行動するようになった |
| ゴブリンがHP15・ATK4では完全に無力 | 2セッションともK/D比0.06〜0.08。退却する前に死亡し「数の暴力」コンセプトが機能しなかった。HP28・ATK7・スポーン10秒に調整済み |
| ゴブリンの数の暴力問題 | スポーン7秒・上限なしで476スポーン382体が同時展開しゲームバランス崩壊。UNIT_CAP_PER_CRYSTAL=5の導入とスポーン10秒延長で解決。K/D 0.45〜0.75の健全な範囲に収まった |
| 旧オーガ（ATK8）がS1で2.78 K/Dと異常に強い | HP60＋ATK8で相性デバフがほぼ効かない構成。ATK7へ引き下げ |
| リザードがS1で7分半、S2で3分で消滅 | HP25では全勢力の最弱クラスで早期脱落。HP45・ATK10に引き上げ後、段階的に調整 |
| リザードATK10→9でも支配的（K/D 0.94〜1.54） | S4〜S8でリザードが複数勝利。HP45→28へ削減後、ATK9→8へ段階調整。現在K/D 0.52〜0.88（人間勝利時） |
| リザードの初動マップ有利によるスノーボール | t=60でクリスタル8個超を確保すると退却回復成功率が80%超に上昇し手がつけられなくなる。マップ運の要因が強く、ステータス単体での解決は難しい（S16事例） |
| エルフATK8がS2で3.33 K/Dと強すぎた | 人間側圧勝時に生存が長くなりキル数が膨らんだ。ATK8→6に調整 |
| エルフHP20が三種ローテの弱点 | 最弱ユニットとして退却成功率が低く、人間族全体の継戦力を下げていた。HP20→25で生存時間が約25%延び改善 |
| 人間大量スポーンで5分ゲームになる | S2でt=90に人間48体が同時展開し三すくみを無視して圧勝。UNIT_CAP_PER_CRYSTAL=5の導入とクールダウン設定で抑制 |
| 人間スポーンCD5秒・上限なしで449体超スポーン | 上限チェックなしで圧倒的優位に。ユニット上限チェック追加後、適切な数に収まる |
| 人間スポーンCD10秒・上限ありで勝率50%→75% | S9〜S12で50%だった人間勝率が、リザードATK8・エルフHP25適用後のS13〜S16で75%（3勝1敗）に改善 |
| 人間スポーンCD12秒で60%目標への調整 | CD10秒時に75%と目標を超過したためCD12秒（20%遅化）に延長。その後、ターゲットアルゴリズム統一に伴い10秒に戻した |
| 退却成功率は50〜75%で適切に機能 | 全種族で退却システムが有効。回復速度MaxHP÷8/秒（約5秒でフル回復）は妥当な値 |
| ドワーフが人間3種の火力主役 | S13〜S15でドワーフのK/Dが1.34〜2.30と3種中最高。ゴブリン・リザードへのキル数で安定したトップ |
| 定期再ターゲット（10〜20秒/50%）の廃止 | クリスタル直前で引き返す不自然な挙動の原因。スポーン時ランダムで分散目的は達成できるため廃止。全陣営で「狙ったら取り切る」挙動に統一 |
| 全陣営のターゲットアルゴリズム統一 | 旧来は人間族のみ最近傍（bestTarget）、他陣営はランダム。人間の強さの源泉が「最短拡張」にあったため、全陣営を距離反比例の重みつきランダムに統一し格差を是正 |
| ランダム統一後にオーガが過剰強化 | HP60の高耐久がランダム長距離移動と相性良く、K/D 3.8〜3.9まで上昇。5陣営中4陣営に優位な相性補正も要因。ATK 7→6に削減 |
| マップ運依存の高分散問題 | t=60のクリスタル数がt=150の結果をほぼ決定。初動60秒のマップ運でゲームが決まる傾向。対策として①距離重みつきターゲット選択（近傍優先化）②拠点近傍への中立クリスタル構造配置（各拠点隣接に2個保証）を導入 |
| 中立クリスタル配置の構造化 | 旧来の中央16個ランダムから、拠点近傍2個×4陣営（8個）＋中央8個に変更。各陣営が序盤に安定して2〜3個取れるようになり初動格差が縮小 |
